# 🌸 КонМари — Моя организация

Веб-приложение для инвентаризации вещей по методу Мари Кондо. Работает прямо в браузере — без установки, без сервера, без регистрации.

## ✨ Возможности

- **Три уровня структуры** — Разделы → Категории → Подкатегории
- **Базовая структура** из коробки (Одежда, Обувь, Аксессуары, Косметика, Документы) + возможность создавать свои
- **Фото и PDF** — прикрепляй фотографии вещей или сканы документов
- **Метод КонМари** — отмечай каждую вещь: ✨ Дарит радость / 🗑 Убрать / 🤔 Решу позже
- **Навигация** — главный экран с крупными плитками, хлебные крошки, подкатегории-табы
- **Фильтры** — по радости, сортировка по дате / алфавиту
- **Всё хранится локально** — данные сохраняются в `localStorage` браузера, ничего не уходит на сервер

## 🚀 Использование

Просто открой `index.html` в браузере. Никаких зависимостей, никакого сборщика.

Или запусти через GitHub Pages — загрузи репозиторий и включи Pages в настройках.

## ⚠️ Ограничения localStorage

Браузер даёт ~5 МБ на домен. Если добавляешь много фото/PDF — следи за местом. При переполнении приложение предупредит и сохранит запись без файла.

**Совет:** для резервной копии открой DevTools → Application → Local Storage → скопируй значение ключа `konmari_v4`.

## 🛠 Технологии

- Чистый HTML + CSS + JavaScript (vanilla, без фреймворков)
- Google Fonts: Cormorant Garamond + DM Mono
- `localStorage` для хранения данных
- `Canvas API` для сжатия фотографий
- `FileReader API` для загрузки файлов
